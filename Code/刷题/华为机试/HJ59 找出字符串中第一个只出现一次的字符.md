## 描述

找出字符串中第一个只出现一次的字符

数据范围：输入的字符串长度满足 $1≤n≤1000$ 

### 输入描述：

输入一个非空字符串

### 输出描述：

输出第一个只出现一次的字符，如果不存在输出-1

## Answer
```python
s = input()
# 创建一个空字典来存储字符以及它们出现的次数
str_dic = {}
# 遍历字符串 s，并统计每个字符出现的次数
for c in s:
    if c not in str_dic:
        str_dic[c] = 1
    else:
        str_dic[c] += 1
# 遍历字符串 s，找到第一个只出现一次的字符并打印出来
for c in s:
    if str_dic[c] == 1:
        print(c)
        break
else:
    # 如果没有找到只出现一次的字符，输出 -1
    print(-1)
```
