## **描述**
输入一个整数，将这个整数以字符串的形式逆序输出
程序不考虑负数的情况，若数字含有0，则逆序形式也含有0，如输入为100，则输出为001
**数据范围：** $0≤n≤2^{30}−1$
### **输入描述：**
输入一个int整数
### **输出描述：**
将这个整数以字符串的形式逆序输出
### **Answer**
```python
# 从用户输入获取一个字符串，并使用切片[::-1]将字符串逆序输出
print(input()[::-1])
```
