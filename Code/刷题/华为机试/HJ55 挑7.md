## 描述
输出 1到n之间 的与 7 有关数字的个数。
一个数与7有关是指这个数是 7 的倍数，或者是包含 7 的数字（如 17 ，27 ，37 ... 70 ，71 ，72 ，73...）
数据范围： $1≤n≤30000$ 
### 输入描述：
一个正整数 n
### 输出描述：
一个整数，表示1到n之间的与7有关的数字个数。
### answer
```python
# 读取输入的整数
n = int(input())
# 初始化结果变量为 0
res = 0
# 遍历从 1 到 n 的每个数
for i in range(1, n + 1):
    # 如果当前数能被 7 整除，或者当前数的字符串表示中包含 '7'，则结果加一
    if i % 7 == 0 or '7' in str(i):
        res += 1
# 输出结果
print(res)
```
